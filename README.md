## Start the project

- `npm install`

- `npm install -g peer`

- Execute the `start.bat` script

## Usage

Open a browser, and access to the `index.html` file

From here, you can generate a room.

From another tab, access to the created room, and start chating !

You can also send a Private Message by using a user ID

## Video calls

Video call feature is in the `webcam` branch. Not fully implemented with other features, but a version is working
