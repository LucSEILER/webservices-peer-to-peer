# peer to peer chat

## Start the project

- `npm install`

- `npm install -g peer`

- `peerjs --port 9000 --path /peerjs`

- `node .\server.js`

## Usage

Open a browser, and access to the `index.html` file

From here, you can generate a room.

From another tab, access to the created room, and start chating !
